## 部署步骤

### 方式一：Vercel 免费部署（推荐）

```bash
# 1. 安装 Vercel
npm i -g vercel

# 2. 部署后端（需要改造成 Serverless）
cd server
vercel deploy
```

### 方式二：国内服务器

推荐配置：
- 服务器：2核4G / 50G SSD
- 系统：CentOS 7+ / Ubuntu 20.04
- Web服务器：Nginx
- Node.js：v18+

```bash
# 服务器部署命令
cd /opt/shenjimiaosuan
git pull
pm2 restart server
nginx -s reload
```
